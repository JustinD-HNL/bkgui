Step Properties

Select a step to view and edit its properties

Click on any step in the pipeline to configure its settings

Available Properties

  • Label: Display name for the step
  • Command: Shell command to execute
  • Key: Unique identifier for dependencies
  • Agents: Agent targeting requirements
  • Environment: Environment variables
  • Plugins: Buildkite plugins to use
  • Dependencies: Step dependencies
  • Conditions: Conditional execution rules
